# vue+vant+eggjs通用前端框架

## quick start
```
h5端
yarn install
eggjs中间接口转发
cd server
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
cd server
npm run dev
```

### Compiles and minifies for production
```
yarn build
cd server
npm start
```

### Lints and fixes files
```
yarn lint
```
